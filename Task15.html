Tool Documentation

While working on this project, I found many exciting tools for data visualisation. I also stumbled on some stunning artwork by people working in this field. First, I learned how to use HTML, CSS and JavaScript and how powerful these tools are for data visualisation. I believe, that creating my portfolio as a website allowed me to tap into a new dimension of interactivity and animation. A central source for this project was Flourish, an organisation based in the UK which provides templates as well as a hosting service for your visualisations. Of course, GitHub which allows you to host your Website over their servers. html5up offers html templates under a creative commons license so you don't have to start from scratch. D3js is a JavaScript library which can be used to create interactive visualisations.

Despite these tools I also discovered some exciting artists such as Lindsey Poulter, Nathan Yau, Kristina Szucs, Kristina Szucs and Cuthbert Chow.
